# Repository instructions

Before changing this project, read `PROJECT_SPEC.md` completely and treat it as the product source of truth.

- Preserve the lightweight scope and the fixed navigation order.
- Keep user data device-local unless `PROJECT_SPEC.md` is intentionally updated.
- Never place API keys or other secrets in source code, Git history, browser assets, or Android artifacts.
- Do not switch the production host back to ChatGPT Sites.
- Verify both the web build and Android behavior after changes that affect deployment or native detection.
